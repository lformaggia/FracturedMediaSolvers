Detailed Description

Class Solver.

This is a base abstract class that implements a linear solver for the system Ax=b. It receives a Eigen Sparse matrix A and a Eigen Vector b. Each derived class employs a different solver.

Solve the linear system.

Solve the linear system Ax=b. The method employed to solve the system depends on the derived class of Solver that calls this method.

Implemented in FVCode3D::EigenBiCGSTAB, FVCode3D::EigenCG, FVCode3D::IterativeSolver, FVCode3D::EigenLU, and FVCode3D::EigenCholesky.
