Detailed Description

Class for assembling a global bulk builder.

This is the classe to assemble the global bulk builder. It's useful using a builder of this type to assemble efficiently the bulk matrices. With this builder we can assemble M, B, Dt matrices jointly with only one cells loop. This builder allow both to build up M, B, Dt separately or to build up these matrices in the monolithic matrix of the system
