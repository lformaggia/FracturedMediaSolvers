void FVCode3D::DarcyPseudoSteady< QRMatrix, QRFracture, MatrixType, Implicit >::initialize ( )
virtual

Initialize the matrices.

Build the matrices and vectors that are not time dependent, i.e., the mass matrix, the stiffness matrix, the source vector and the BCs. The BCs and the source are not added to the RHS. To be called only once.
