Detailed Description

Base class for assembling a global operator of the problem.

This is the base class to build one of the operators of the problem (the operator can be a mimetic one associated to the bulk problem or a fracture operator such as the coupling conditions or the finite volume trasmissibility matrix). It's an abstract class.
