Detailed Description

Class for assembling a coupling conditions matrix.

This is the classe to assemble the finite volume transmissibility matrix (changed in sign) in fracture. To do it a fracture loop is performed; to take into account the fractures intersection the "star-delta" transformation is employed. This class allow also the imposition of bcs on fractures.
