Detailed Description

Class that implements method to integrate function over the mesh.

This class integrates solutions to PDE solved with the finite volume method, computes the L2Norm of such a solution and integrates continuous functions. By passing a QuadratureRule to the constructor it is possible to choose the quadrature rule you want to use.
